# Employee Management System

Spring Boot application with PostgreSQL database integration and JWT Authentication.

## Features
- User Registration
- User Login with JWT
- Employee CRUD Operations
- Pagination and Sorting
- Spring Security
- JPA/Hibernate
- PostgreSQL Database

## Technologies Used
- Java 17
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Maven
- Hibernate/JPA
- Postman

## API Testing
Use Postman to test endpoints.

## Run Project

```bash
mvn spring-boot:run
